# Tea Catalog API

Full-stack tea catalog application.

## Backend

- Django
- Django REST Framework
- JWT Authentication
- Filtering
- Search
- Ordering
- Image Upload

## Frontend

- React
- Vite
- Axios
- React Router

## Features

- User Registration
- Login
- JWT Authentication
- CRUD Operations
- Tea Image Upload
- Search and Filtering
- PostgreSQL
- Cloudinary image storage
- Refresh token handling
- Deployment
- Custom permissions (tea owner only)
- CI/CD
